ls : 리스트 보여줌
cd : change direction 경로 설정
cd.. : 설정한 경로 나가기
pwd : Print Working Directory 작업하고 있는 폴더의 경로를 출력
mkdir : makd directory 디렉토리를 만듦
rmdir : remove directory 디렉토리 지움
echo : 문자열이나 객체를 화면에 출력해줌
ls -al : 숨겨진 파일까지 다 보여줌
rm : 파일 싹 다 삭제

깃허브에 올리는 순서
git add . : 폴더 안에 있는 모든 파일들을 담는다.
git add . 파일명: 파일 하나만 담는다.
git commit -m "메시지" : commit을 해야 포장이 된 거임. 따옴표 안에 글이 무조건 있어야 함.
git push origin main

git push origin main : 깃허브에 올리기
git pull origin main : 깃허브에서 불러오기

<제어문>
조건문 : if 문, if - else 문
반복문 : for 문, while 문
반복문 흐름 변경 : break 문, continue

-조건문
if : 조건 적용
else if (elif) : else 중에서 추가적인 조건 적용
else : if, elif 말고 다

-반복문
for i in range() : range 안에서 i를 받아와라.
range()에서 괄호 안의 순서 (start, stop, step) 시작, 멈춤, 간격
※ i는 단순 매개변수라서 다른 기호를 사용해도 상관 없다.

-리스트 변수
score = [87,84, 95, 67]
for i in score(리스트 변수)를 넣으면 리스트를 프린트 해준다. 숫자 및 문자로도 받을 수 있다.
list(range(5)) : range(5)를 list로 바꾼다.
